DOM full form is Document Object Model. It is used to get the element by id name / class name so we can change the Background-Color, TextColor, Structure, Style, content, and many more interesting work can be done using DOM.

There are many methods to select the html element. Such as using getElementsByTagName() here we have to set the Html tag name, getElementsByClassName() here we have to set the class name under tag name, getElementById() here we have to set the particular Id name we set on tag name, querySelector() it is basically used to select 1st element with the set matched class name, querySelectorAll() it select all element with the all class name which is macthed.

Event Bubble is process / circulation which start from document in html and end on the root area. It works like a tree we can compare top of the tree with document and root of the tree with the targeted area.

CallBack function is one kind of function. Using Callback function one function can get another function code access to make it workable. So, I will use it to make the function working.
